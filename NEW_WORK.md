# New work statement

Every line of this repository was written during **Built with Claude: Life Sciences**
(Jul 7–13, 2026), from scratch, using Claude Code.

- Uses only public datasets and open-source Python dependencies:
  - Marson CD4+ T-cell CRISPRi Perturb-seq (Zhu, Dann, … Marson 2025), MIT-licensed.
  - Replogle genome-scale Perturb-seq in K562 (Replogle et al., Cell 2022, PMID 35688146),
    for the cross-cell-type verifier transfer. Figshare article 20029387.
  - Literature citations resolved through PubMed.
- No pre-existing project code, no imports from any prior codebase.
- Prior expertise in reproducibility and verification informed *what* to build; none of the
  implementation is reused. Commit history is granular and timestamped within the event window.
